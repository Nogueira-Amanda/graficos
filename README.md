# Projeto de Gráficos com Matplotlib:
Esse projeto foi desenvolvido com o objetivo de praticar o uso da biblioteca Matplotlib em python para a criação de gráficos.

# Objetivos:
- Representar visualmente os dados de um sistema de estoque usando gráficos de linhas, barra, setorial (pizza) e dispersão.
- Analisar tendências e comparações de estoque e produtos.

# Exercícios:
1 - Tendência de Estoque:
Esse gráfico em linha tem o objetivo de demonstrar a evolução de um produto ao longo de uma semana.

2 - Comparação de Produtos:
Esse gráfico de barras compara as quantidades de produtos existentes de diferentes categorias.

3 - Proporção de Categorias:
Esse gráfico setorial (comumente chamado gráfico de pizza) se presta a demonstrar a proporção do valor total em estoque por categoria de produto.

4 - Preço x Qualidade:
Esse gráfico relaciona o preço unitário e a quantidade em estoque.

# Como executar:
1. Tenha uma ide que rode python;
2. Instale a extenção pip install matplotlib;
3. Execute o arquivo.
